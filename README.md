Анализ маркетинговой кампании португальского банка
Описание проекта
В данном проекте проводится исследовательский анализ данных (EDA) прямой маркетинговой кампании португальского банковского учреждения. Основная цель — изучить профиль клиента и определить факторы, которые влияют на принятие решения об открытии срочного депозита.

Стек технологий
Язык: Python

Библиотеки: Pandas, Matplotlib, Seaborn

Основные этапы работы
Предобработка данных: Загрузка и проверка структуры датасета (11,162 записи), контроль пропусков и типов данных.

Сегментация: Разделение клиентской базы по возрастным группам (от 18 до 100 лет) для детального анализа.

Исследовательский анализ (EDA):

Анализ влияния социально-демографических признаков (образование, работа, семейное положение).

Изучение финансовых показателей (баланс, наличие кредитов).

Оценка эффективности каналов связи и длительности контактов.

Визуализация: Построение графиков распределения и сравнительных диаграмм (Box Plots, Heatmaps, Count Plots) для выявления закономерностей.

Ключевой вывод
В ходе анализа было выявлено, что наличие предыдущего успешного контакта с клиентом (pdays) является одним из решающих факторов: в сегменте клиентов, с которыми банк уже взаимодействовал ранее, наблюдается значительно более высокая конверсия в открытие депозита.

Как запустить
Клонируйте репозиторий.

Убедитесь, что установлен Python и необходимые библиотеки (pip install pandas matplotlib seaborn).

Откройте Project.ipynb через Jupyter Notebook или VS Code.
