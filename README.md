# Анализ маркетинговой кампании португальского банка
***
## Описание проекта
В данном проекте проводится исследовательский анализ данных (EDA) прямой маркетинговой кампании португальского банковского учреждения. Основная цель — изучить профиль клиента и определить факторы, которые влияют на принятие решения об открытии срочного депозита.
***
## Стек технологий
* Язык: Python
* Библиотеки: Pandas, Matplotlib, Seaborn
***
## Основные этапы работы
* __Предобработка данных__: Загрузка и проверка структуры датасета (11,162 записи), контроль пропусков и типов данных.
* __Сегментация__: Разделение клиентской базы по возрастным группам (от 18 до 100 лет) для детального анализа.
* __Исследовательский анализ (EDA)__:

    * Анализ влияния социально-демографических признаков (образование, работа, семейное положение).

    * Изучение финансовых показателей (баланс, наличие кредитов).

    * Оценка эффективности каналов связи и длительности контактов.
* __Визуализация__: Построение графиков распределения и сравнительных диаграмм (_Heatmaps, Count Plots_) для выявления закономерностей.
***
## Ключевой вывод
В ходе анализа было выявлено, что наличие предыдущего успешного контакта с клиентом (_pdays_) является одним из решающих факторов: в сегменте клиентов, с которыми банк уже взаимодействовал ранее, наблюдается значительно более высокая конверсия в открытие депозита.
